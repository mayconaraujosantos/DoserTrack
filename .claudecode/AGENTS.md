# Doser Specialist Agents

## Backend/Database Agent

**Purpose**: Manage SQLite database operations, migrations, queries, and data layer  
**Expertise**:

- Database schema design and migrations
- Query optimization for SQLite
- Data synchronization logic
- Local storage patterns

**When to invoke**: Database operations, query debugging, sync issues
